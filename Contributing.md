# Contributing to Intelex

## Getting Started

1. Clone the repository.
2. Navigate to the frontend folder.
3. Install dependencies.

```bash
npm install
npm run dev
```

---

## Before You Start

Please read:

- README.md
- docs/Software_Requirements_Specification.md
- docs/Frontend_Design.md
- docs/ui_guidelines.md
- docs/Task_Allocation.md

---

## Branching

Do not work directly on `main`.

Create a new branch:

```bash
git checkout -b feature/your-feature-name
```

Example:

- feature/case-book
- feature/speech-to-text
- feature/document-generator

---

## Commits

Use meaningful commit messages.

Examples:

- Build case book page
- Add speech recording UI
- Refactor shared input component

---

## Pull Requests

- Push your branch.
- Create a Pull Request to `main`.
- Wait for review before merging.

---

## Guidelines

- Follow the UI Guidelines.
- Reuse shared components.
- Keep code clean and organized.
- Ask before changing the project structure.
