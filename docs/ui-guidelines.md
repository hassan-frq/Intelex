# Intelex UI Guidelines

# Typography

- Font Family: Inter
- Heading: Bold
- Body Text: Regular
- Labels: Medium

---

# Color Palette

## Backgrounds

- App Background: `bg-zinc-950`
- Cards: `bg-zinc-900`
- Sidebar: `bg-zinc-900`

## Text

- Primary: `text-white`
- Secondary: `text-zinc-400`
- Labels: `text-zinc-300`

## Borders

- `border-zinc-800`

## Primary Action

- Button: `bg-blue-600`
- Hover: `bg-blue-700`

---

# Border Radius

- Cards: `rounded-2xl`
- Buttons: `rounded-xl`
- Inputs: `rounded-xl`

---

# Spacing

- Use generous whitespace.
- Avoid crowded layouts.
- Maintain consistent spacing between sections.

---

# Icons

- Library: React Icons
- Preferred Icon Set: Feather Icons (`react-icons/fi`)

---

# Components

## Button

- Full width where appropriate.
- Primary color: Blue.
- Rounded corners.
- Hover effect enabled.

## Input

- Rounded corners.
- Dark background.
- Clear focus state.
- Labels above inputs.

## Card

- Dark background.
- Rounded corners.
- Subtle shadow.
- Consistent padding.

## Scrollbars

Use a dark scrollbar throughout the application.

- Track: `zinc-900`
- Thumb: `zinc-600`
- Hover: `zinc-500`

Scrollbar styling should remain consistent across all pages.

---

# Layout

## Authentication

- Centered card.
- Minimal design.
- No sidebar.

## Main Application

- Fixed Sidebar.
- Top Navbar.
- Content area with consistent padding.

---

# General Rules

- Keep the interface minimal.
- Reuse shared components whenever possible.
- Do not hardcode colors.
- Maintain consistency across all pages.
- Mobile responsiveness should be considered during development.
